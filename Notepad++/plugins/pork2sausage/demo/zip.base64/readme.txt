This is a demo program made in java for Pork2Sausage (un Notepad++ plugin).
The program zip or unzip a given string from stdin then put the result to stdout.

Example to zip:
java -classpath C:/MY_PATH/zip.base64;C:/MY_PATH/zip.base64/commons-codec-1.4.jar zipB64 -zip "This is the text to be compressed"

Example to unzip:
java -classpath C:/MY_PATH/zip.base64;C:/MY_PATH/zip.base64/commons-codec-1.4.jar zipB64 -unzip eNoLycgsVgCikoxUhZLUihKFknyFpFSF5PzcgqLU4uLUFADH7gwa
